# Task-3: Temperature Monitoring System using I2C LCD

## Internship
CodTech Technologies – Embedded Systems Internship

## Objective
To design a temperature monitoring system using a DS18B20
digital temperature sensor and display the readings on a
16×2 I2C LCD.

## Components Used
- Arduino UNO
- DS18B20 Temperature Sensor
- 16×2 I2C LCD
- 4.7kΩ Pull-up Resistor
- Jumper Wires

## Working Principle
The DS18B20 sensor measures temperature and communicates
with Arduino using the OneWire protocol. The Arduino
processes the temperature data and displays it on the
I2C LCD in real time.

## Output
The LCD displays live temperature values in degree Celsius.

## Platform Used
Wokwi Arduino Simulator
