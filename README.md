# Task-1: Temperature Monitoring using DS18B20

## Description
This project is part of the CodTech Embedded Systems Internship.
It measures ambient temperature using a DS18B20 digital temperature
sensor and displays the output on the Arduino Serial Monitor.

## Components Used
- Arduino UNO
- DS18B20 Temperature Sensor
- 4.7kÎ© Pull-up Resistor
- Jumper Wires

## Working
The DS18B20 communicates via the OneWire protocol and provides
digital temperature readings directly to the microcontroller.

## Output
Temperature values are displayed every second on the serial monitor.

## Platform Used
Wokwi Arduino Simulator
